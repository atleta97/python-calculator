# Python Calculator

This project was developed as part of my transition into scientific programming and software development. It demonstrates fundamental Python programming concepts through a modular command-line calculator.

The objective was to strengthen programming fundamentals while following good coding practices and preparing for more advanced computational engineering projects.

- Modular function design

- Clean and readable code structure
